# e-plantShopping - Paradise Nursery

Welcome to **e-plantShopping**, a React-based web application designed for a seamless houseplant shopping experience. This project demonstrates the implementation of a professional shopping cart using **Redux Toolkit** for state management.

## Project Overview
This application, titled **Paradise Nursery**, allows users to:
* Browse a diverse catalog of air-purifying, aromatic, and medicinal plants.
* Add plants to a shopping cart with real-time feedback.
* Manage the cart by increasing, decreasing, or removing items.
* View total costs and item counts dynamically.

## Technologies Used
* **React**: Frontend library for building the UI components.
* **Redux Toolkit**: For centralized state management (CartSlice).
* **Vite**: For a fast and modern development environment.
* **GitHub Pages**: For live application deployment.
